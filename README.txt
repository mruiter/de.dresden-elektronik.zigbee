Let Homey control LED drivers manufactured by Dresden Elektronik.

It currently only works with the fls-ct-lp type, but more controllers might be added in the future.
The fls-ct-lp is the same controller as the Hera LED 24 V Dynamic-Controller, productnumber 615 000 514 11.

Note that this app will not work with other products manufactured by Dresed Elektronik.
